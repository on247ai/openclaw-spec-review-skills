# Subagent Review Playbook

## Khi spawn reviewer
- artifact quan trong
- can phan bien doc lap
- can 2-3 goc nhin khac nhau

## Prompt reviewer nen co
- artifact dang review la gi
- scope review
- standard severity
- verdict can tra ve: approve / revise
- khong sua artifact goc, chi review

## Sau khi co review
- tong hop finding
- xu ly blocker truoc
- quyet dinh co can vong review tiep hay khong
