# RFC Decision Tree

Cần RFC nếu trả lời CÓ cho ít nhất 1 câu:
- Có thay đổi kiến trúc lớn không?
- Có từ 2 phương án khả thi trở lên không?
- Có migration / rollback / data integrity risk không?
- Có ảnh hưởng nhiều module hoặc nhiều team không?
- Có security / privacy / cost impact đáng kể không?
- Có ambiguity cao cần debate trước khi chốt không?

Nếu tất cả là KHÔNG, thường có thể đi thẳng vào spec.
